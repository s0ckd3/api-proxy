## Login API:

- Url: https://1proxy.net/api/auth/login

- Json body: 
```
{
	"email": "username@gmail.com",
	"password": "password"
}
```

- Response:
```
{
  "user": {
    "id": 5,
    "first_name": "user",
    "last_name": "name",
    "phone_number": "0999999999",
    "email": "username@gmail.com",
    "role": "MEMBER",

    "updated_at": "2023-11-04T05:48:38.000000Z"

  "token": "your-token",
  "token_type": "Bearer"
}
```
